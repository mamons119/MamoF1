🏎️ MamoF1

MamoF1 è un'applicazione che permette di esplorare, analizzare e confrontare le statistiche e i dati di telemetria dei piloti dell’ultima sessione di Formula 1 (prove libere, qualifica, sprint race e gara).

✨ Funzionalità
- 📊 Visualizzazione delle statistiche e telemetria dei piloti
- 🆚 Confronto diretto tra più piloti
- 📡 Analisi della telemetria post-gara
- 📈 Grafici interattivi (velocità, acceleratore, freno, RPM, ecc.)
- ⏱️ Confronto tempi sul giro e settori
- 🎧 Riproduzione dei team radio dei piloti
- 🏁 Caricamento automatico dell’ultima sessione disponibile
- 🗺️ Visualizzazione delle statistiche del circuito dell’ultima sessione

🌐 Disponibilità
MamoF1 è disponibile in due versioni:
- 💻 Web App – accessibile direttamente da browser
- 📱 Android (APK) – installabile manualmente su dispositivi Android
- 
MamoF1 è stata sviluppata usando l'engine Construct 3 e sfrutta le API pubbliche di OpenF1 per fornire dati dettagliati e aggiornati direttamente dalle sessioni ufficiali.
