🏎️ MamoF1

MamoF1 è un'applicazione che permette di esplorare, analizzare e confrontare le statistiche e i dati di telemetria dei piloti dell’ultima gara di Formula 1.

✨ Funzionalità
- 📊 Visualizzazione delle statistiche dei piloti
- 🆚 Confronto diretto tra più piloti
- 📡 Analisi della telemetria post-gara
- 📈 Grafici interattivi (velocità, acceleratore, freno, RPM, ecc.)
- ⏱️ Confronto tempi sul giro e settori
- 🏁 Caricamento automatico dell’ultima gara disponibile

L’app è stata sviluppata usando l'engine Construct 3 e sfrutta le API pubbliche di OpenF1 per fornire dati dettagliati e aggiornati direttamente dalle sessioni ufficiali.
